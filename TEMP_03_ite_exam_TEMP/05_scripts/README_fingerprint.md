# ITE Exam Fingerprint Clustering

NLP-based clustering of 1,200 ABFM ITE question stems (2020–2025) to surface
**hidden exam archetypes** — *how* the board tests, not just *what* it tests.

---

## Purpose

The existing taxonomy (BodySystem / Subcategory) answers: *"What topic is being tested?"*  
This pipeline answers: *"What type of cognitive task is ABFM actually asking?"*

That distinction matters for targeted study. A question tagged `Cardiovascular / Pharmacology`
could be asking you to:
- **Select a first-line drug** (Pharmacology archetype)
- **Interpret an EKG and act on it** (Diagnostic Workup archetype)
- **Counsel a patient on side effects** (Counseling archetype)

Each requires a different mental skill. Clustering surfaces which archetypes dominate
the exam and how their prevalence has shifted 2020→2025.

## The 12 Archetypes (defaults)

Archetype names are **heuristic** — generated by keyword rules in `infer_archetype()`.

| Archetype | Typical stem pattern |
|-----------|----------------------|
| Acute / Emergent Presentation | "comes to ED with sudden…" |
| Pediatric Care & Development | "mother brings 4-year-old…" |
| Chronic Disease Management | "A1C is 8.2, currently on metformin…" |
| Pharmacology & Drug Selection | "Which medication is first-line for…" |
| Screening & Prevention Guidelines | "USPSTF recommends screening for…" |
| Diagnostic Workup & Lab Interpretation | "Labs show TSH 0.1; next step?" |
| Psychiatric & Behavioral Health | "Patient reports depressed mood for 3 weeks…" |
| Musculoskeletal & Pain | "Knee pain worsens with stairs…" |
| Obstetric & Perinatal Care | "32-week pregnant patient presents…" |
| Women's Health (Non-OB) | "LMP 14 months ago, hot flashes…" |
| Geriatric & Functional Medicine | "76-year-old with cognitive decline…" |
| General Adult Medicine — Undifferentiated | Catch-all |

### Customizing archetype names
1. Run the script once
2. Open `fingerprint_cluster_summary.csv`
3. Review `TopTerms` for each cluster
4. Edit the keyword rules in `infer_archetype()` to better match what you see
5. Re-run — labels update automatically

## Tuning parameters (top of script)

| Variable | Default | Effect |
|----------|---------|--------|
| `N_CLUSTERS` | 12 | Number of archetypes; try 8–16 |
| `TFIDF_FEATS` | 2000 | Vocabulary size |
| `SVD_COMPS` | 100 | LSA dimensions before UMAP |
| `UMAP_NEIGH` | 20 | UMAP neighborhood size (larger = more global) |
| `UMAP_MINDIST` | 0.08 | UMAP cluster tightness |
| `RANDOM_SEED` | 42 | Reproducibility |

## Integration with main pipeline

After running, merge `fingerprint_labels.csv` into `ABFM_ITE_Master_v2.xlsx`
to add `ArchetypeName` as a new column. This can then be used by downstream
enrichment scripts to tag the `BoardPrep-ContentOutline_HY-ENRICHED` document
with archetype metadata per ITE question block.
