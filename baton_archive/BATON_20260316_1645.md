## Key Architecture Reminders

- **Codon format:** `Author_Year#@#ART-XXXX@#@.pdf` — `#@#` = start, `@#@` = stop
- **Pipeline order:** convert_pdfs_to_json.py → ite_intelligence_enricher_batch.py → build_crosswalk_index.py
- **Baton lifecycle:** archive → sync docs → write new (always in this order)
- **Baton naming:** active = `BATON_active_YYYYMMDD_HHMM.md` at root; archived = `BATON_YYYYMMDD_HHMM.md` in baton_archive/
- **Enriched JSON folder:** `03_enriched_JSON` (underscore, no space)
- **Enrichment v4:** codon_filename → high confidence; clean_ref → high confidence; no other strategies
- **Trend queries:** Always use `body_system_merged` (not `body_system`) for longitudinal analysis — 2024 taxonomy change
- **Join paths:** `question_ref_pairs` (canonical, uses clean_ref) or `qid_art_xref` (ergonomic, uses article_id)
- **ICD-10 join chain:** `icd10_rollup` → `icd10_code_xref` → `article_icd10` → `articles` → `qid_art_xref` → `questions`
- **Standard filters:** Exclude `citation_count = 0`, `source_type = 'stub'`, `article_id = 'ART-0001'`
